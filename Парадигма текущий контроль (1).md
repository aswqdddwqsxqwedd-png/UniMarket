UniMarketС - это платформа для студентов университета, где можно:
- Продавать и покупать товары (учебники, электроника, мебель)
- Общаться с продавцами через чат
- Искать товары по категориям
- Администраторы могут управлять контентом
Технологический стек
Backend: FastAPI (Python 3.10+)
Database: PostgreSQL + SQLAlchemy
Testing: pytest
Tools: Git, Docker, pre-commit
Структура проекта
UniMarket/
├── src/                    # Основной код приложения
## │   ├── __init__.py
│   ├── main.py            # Точка входа FastAPI
│   ├── config.py          # Конфигурация
│   ├── models/            # SQLAlchemy модели
│   ├── schemas/           # Pydantic схемы
│   ├── routers/           # API endpoints
│   └── utils/             # Вспомогательные функции
├── tests/                 # Тесты
├── alembic/              # Миграции БД
├── .env                  # Переменные окружения
## ├── .gitignore
├── requirements.txt
└── README.md
## Структура-пояснение
- src/ - весь код приложения в одной папке (легко импортировать)
- tests/ - тесты отдельно от кода
- alembic/ - история изменений базы данных
- .env - секреты не попадут в Git

Создаем файл src/config.py:
# src/config.py

## """
Конфигурация приложения UniMarket
## """
from pydantic_settings import BaseSettings
from typing import Optional

class Settings(BaseSettings):
"""Настройки приложения"""

# Основные настройки
PROJECT_NAME: str = "UniMarket"
VERSION: str = "0.1.0"
DEBUG: bool = True
# API настройки
API_PREFIX: str = "/api/v1"

class Config:
env_file = ".env"
case_sensitive = True

# Создаем глобальный объект настроек
settings = Settings()
